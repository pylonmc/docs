
This page lists all the placeholders that Rebar supplies when [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) is installed.

!!! hint "Testing placeholders"
    You can see how placeholders look in-game by using the `/papi parse me <placeholder>` command.

| Placeholder                        | Description
|------------------------------------|------------------------------------------|
| `%rebarwaila_text%`                | Shows the player's current WAILA text in MiniMessage format
| `%rebarwaila_plain_text%`          | Shows the player's raw current WAILA text
| `%rebarwaila_color%`               | Shows the player's current WAILA color
| `%rebarwaila_overlay%`             | Shows the player's current WAILA overlay (i.e. type of bossbar)
| `%rebarwaila_progress%`            | Shows the player's current WAILA bossbar progress
